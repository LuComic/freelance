# Revision Requests

This page is for structured change requests during the design and build process.

## How To Submit A Good Revision Request

Please be specific.

Good example:
Change the homepage hero subheading so it speaks more directly to small business owners, and simplify the final sentence.

Weak example:
Make it better.

## Suggested Feedback Tags

- Copy
- Layout
- Visual
- Mobile
- Desktop
- Functionality
- Bug
- New idea
- Nice-to-have

## Priority Guidance

When leaving feedback, try to make it clear whether something is:

- essential before launch
- important but not blocking
- optional polish

## Notes

This page is best used for active revision rounds, while major strategy questions should stay on their relevant pages.
