# Progress Board

This page gives a live overview of where the project stands.

## Recommended Board Purpose

Use the board to make work visible without needing separate status update messages.

## Typical Task Types

- Homepage design
- Services page layout
- About page copy
- Contact form setup
- Mobile polish
- SEO basics
- Analytics setup
- Launch checklist

## Suggested Tag Types

- Waiting for client
- Design
- Content
- Development
- Revision
- Launch

## Status Meaning

Suggested kanban interpretation:

- Todo: not started or waiting
- In Progress: currently being worked on
- Done: completed or approved

## Notes

If something is blocked by missing input from the client, label it clearly so the blocker is visible.
