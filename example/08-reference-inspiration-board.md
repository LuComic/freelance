# Reference / Inspiration Board

This page collects examples that help shape the visual and structural direction.

## What To Add Here

Please share websites, sections, styles, or UI ideas that feel relevant to this project.

## Useful Comment Format

When sharing a reference, try to include:

- what you like
- what you do not like
- whether it is about layout, typography, color, structure, or tone

## Reference Categories

Suggested idea or feedback tags:

- Homepage inspiration
- Typography
- Color
- Navigation
- Service page structure
- Portfolio style
- Contact page
- Mobile feel
