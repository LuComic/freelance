These files are written to map well to the current Pageboard component set:

- Text components for headings and guidance
- `Radio` for one-choice decisions
- `Select` for multi-choice preferences
- `Feedback` for open-ended requests and notes
- `IdeaBoard` for collaborative ideas and prioritization
- `Kanban` for progress visibility
- `Calendar` for milestones and deadlines
- `PageLink` for navigation between pages
